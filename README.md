## 项目简介
漫画阅读平台，以Vue.js作为前端框架，Flask作为后端框架，并使用Python来抓取数据。
之所以开发壹本漫画，出于以下几个目的
- 完成毕业设计
- 满足自己日常浏览漫画的需求
- 加深自己对于Vue.js前端框架、Flask后端框架、以及Python爬虫的理解
